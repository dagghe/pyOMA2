Installation
============

Install the core (headless) library with pip:

.. code:: console

   pip install pyOMA-2

For the full desktop experience (interactive plots and 3D visualization), install
the optional extras:

.. code:: console

   pip install pyOMA-2[all]

``[plot]`` adds Matplotlib plotting only, ``[3d]`` adds the PyVista/Qt 3D viewer, and
``[all]`` adds both.

Or install with conda/mamba (full install):

.. code:: console

   conda install pyOMA-2

You'll probably need to install **tk** for the GUI on your system, here some instructions:

* `Windows <https://www.pythonguis.com/installation/install-tkinter-windows/>`_

* `Linux <https://www.pythonguis.com/installation/install-tkinter-linux/>`_

* `Mac <https://www.pythonguis.com/installation/install-tkinter-mac/>`_
