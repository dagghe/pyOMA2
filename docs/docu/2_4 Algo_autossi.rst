The ``aoma`` algorithm module
=============================

This module provide the users with automated algorithms for operational modal analysis and modal parameter estimation.

Classes:
   :class:`.AutoSSI`
      Implements the AutoSSI algorithm for single setup.

.. Note::
   Users should be familiar with the concepts of modal analysis and system identification to effectively use this module.
